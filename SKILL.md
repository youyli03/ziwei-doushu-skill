# 紫微斗数排盘与解读

## 概述

本 skill 通过在仓库目录执行 `npx tsx cli.ts` 进行排盘，算法层输出完整星曜/宫位/格局/大限数据，AI 再基于内置知识库进行语义解读。

仓库地址：https://github.com/youyli03/ziwei-doushu-skill

## 知识库说明

本 skill 内置三层知识库，排盘完成后 AI 应主动调取作为解读依据：

### 1. 宫位知识库（`lib/ziwei/palace-knowledge.ts`）
- 十二宫（命宫、兄弟、夫妻、子女、财帛、疾厄、迁移、仆役、官禄、田宅、福德、父母）主星断语
- 每颗星在每个宫位的**核心意象、吉象、凶象、倪师语录**
- AI 解读时应引用对应宫位+主星的断语，而非凭空发挥

### 2. 合命知识库（`lib/ziwei/heming-knowledge.ts`）
- 十四主星的**性格特质、配偶描述、婚配建议**
- 夫妻宫断语：薪水族、感情维系、婚姻风险等均来源于此
- 读取方式：取命宫/夫妻宫主星，匹配对应条目

### 3. 古籍原典库（`lib/classics/`）
- 收录《骨髓赋》《紫微全集》《紫微全书》三部古籍
- 提供按星曜名搜索的 `searchByKeyword(keyword)` API
- 排盘后主动搜索命宫主星，将相关原文附于解读末尾

## 分工说明

| 层次 | 负责方 | 内容 |
|------|--------|------|
| 排盘算法 | CLI（`cli.ts`） | 十二宫星曜布局、四化、格局匹配（1100+ 条）、大限顺序 |
| 宫位解读 | AI + 宫位知识库 | 各宫主星含义、吉凶判断 |
| 人物性格/婚配 | AI + 合命知识库 | 配偶描述、性格特质、婚姻建议 |
| 古籍佐证 | AI + 古籍原典库 | 相关古文引用，增强解读权威性 |

## 安装与运行

```bash
git clone https://github.com/youyli03/ziwei-doushu-skill.git
cd ziwei-doushu-skill
npm install
npx tsx cli.ts --year 1990 --month 8 --day 15 --hour 3 --gender male
```

返回 `{ chart, patterns, summary }` 完整结构，可做进一步解析。

## 解读流程（AI 标准操作）

1. 收集用户出生信息（公历年月日、时辰、性别）
2. 执行 CLI 排盘，获取星曜/宫位/格局/大限
3. 从**宫位知识库**取命宫、夫妻宫、财帛宫等关键宫位断语
4. 从**合命知识库**取命宫主星性格特质与配偶描述
5. 从**古籍原典库**搜索命宫主星相关古文
6. 综合以上三层，给出自然语言解读（不要直接粘贴原始断语）
7. 大限分析：结合当前年龄确认所在大限宫，预判近十年走势

## 注意事项

- 用户说"几点出生"需先换算为时辰序号
- 性别影响大限顺序（男顺/女逆），必须确认
- 排盘前请先确认用户提供的是公历还是农历日期
- 单颗星的断语需结合三方四正（对宫、三合宫）综合判断，不可孤立定论
- 化忌、四煞（擎羊、陀罗、火星、铃星）是否入宫是判断凶象的关键条件
