# 紫微斗数排盘

## 概述

调用本地 CLI 脚本生成完整紫微斗数命盘，包括：十二宫星曜、年干四化、格局识别（1100+ 条规则）、大限推算。

## 脚本路径

```
/home/lyy/ziwei-doushu-skill/cli.ts
```

## 用法

### 文本排盘（推荐直接发给用户）

```bash
cd /home/lyy/ziwei-doushu-skill && npx tsx cli.ts \
  --year <公历年> --month <月> --day <日> \
  --hour <时辰序号0-11> --gender <male|female> \
  [--name <姓名>] [--format text]
```

|------|------|--------|
| 0 | 子 | 23:00–01:00 |
| 1 | 丑 | 01:00–03:00 |
| 2 | 寅 | 03:00–05:00 |
| 3 | 卯 | 05:00–07:00 |
| 4 | 辰 | 07:00–09:00 |
| 5 | 巳 | 09:00–11:00 |
| 6 | 午 | 11:00–13:00 |
| 7 | 未 | 13:00–15:00 |
| 8 | 申 | 15:00–17:00 |
| 9 | 酉 | 17:00–19:00 |
| 10 | 戌 | 19:00–21:00 |
| 11 | 亥 | 21:00–23:00 |

### JSON 输出（需要进一步处理时）

```bash
cd /home/lyy/ziwei-doushu-skill && npx tsx cli.ts \
  --year 1990 --month 8 --day 15 --hour 3 --gender male --format json
```

## 注意事项

- 用户提供出生时间时，若只说"几点"而非时辰，需先转换为时辰序号
- 若用户未提供出生时间，询问后再排盘
- 性别影响大限顺序（男顺/女逆），必须确认
- 命宫分析字段可能为空，以格局和十二宫内容为主输出
